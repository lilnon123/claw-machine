## 🛠 Features

- 🕹 ระบบคีบตุ๊กตาแบบลากลงแล้วปล่อย
- 🧸 ตุ๊กตาหลากหลายแบบ รวมถึงตุ๊กตาลับ (Secret)
- 💬 Popup แจ้งเมื่อคีบครบทุกตัว พร้อมให้เริ่มใหม่
- 📱 Responsive ใช้งานได้ทั้งในมือถือและ PC
- 💖 สีชมพูน่ารัก สไตล์แฟนชอบ
